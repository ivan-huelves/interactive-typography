### Descripción
Este bloque reúne propuestas experimentales desarrolladas en el ámbito académico que, debido a su naturaleza de prototipo exploratorio, se presentan conjuntamente para analizar la capacidad de la tipografía para materializar el fenómeno sonoro. Aunque cada proyecto aborda una intención comunicativa distinta todos comparten una base metodológica común centrada en establecer una conexión en tiempo real entre el audio que actúa como *input* y la transformación del glifo que funciona como *output* empleando para ello herramientas como p5.js en el caso de Grix o la combinación de tipografía variable y JavaScript en el resto de propuestas como se exploró en el prototipo número cuatro.

Esta propuesta plantea una tipografía interactiva capaz de reaccionar a diversos aspectos musicales en función del género. La fuente se articula sobre cuatro ejes de variación (peso, ancho de carácter, contraste y ángulo) que modifican su morfología en respuesta a parámetros como el ritmo o el volumen. El proyecto surge con el objetivo de democratizar el diseño gráfico para músicos emergentes, funcionando como el germen de una herramienta web generativa que permite crear cartelería y contenido para redes sociales derivado directamente de las pistas de audio. A nivel técnico, la investigación resultó clave para resolver el reto de la detección del tempo, integrando la librería Web Audio API de Mozilla.

La propuesta establece relaciones directas centradas exclusivamente en el audio, empleando una función normalizadora encargada de cuantificar magnitudes físicas para adaptarlas al espacio de diseño de la fuente. A partir de estos datos, la función significativa opera bajo lógicas de reactividad, vinculando la intensidad del estímulo con la modificación de diversos ejes de variación. Esta arquitectura confirma el requisito de expandir los procesos de significación, transformando la tipografía estática en una entidad performativa capaz de hacer tangible el ritmo y las dinámicas sonoras de la composición musical.
