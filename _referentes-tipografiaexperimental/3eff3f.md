### Descripción
Este bloque reúne propuestas experimentales desarrolladas en el ámbito académico que, debido a su naturaleza de prototipo exploratorio, se presentan conjuntamente para analizar la capacidad de la tipografía para materializar el fenómeno sonoro. Aunque cada proyecto aborda una intención comunicativa distinta todos comparten una base metodológica común centrada en establecer una conexión en tiempo real entre el audio que actúa como *input* y la transformación del glifo que funciona como *output* empleando para ello herramientas como p5.js en el caso de Grix o la combinación de tipografía variable y JavaScript en el resto de propuestas como se exploró en el prototipo número cuatro. #3EFF3F plantea una tipografía variable diseñada para la identidad de un festival de música explorando la interacción usuario-tipografía a través de la captación de movimiento y sonido. Tomando como base estructural la fuente *Cascadeur* el proyecto desarrolla ejes de variación de expansión y cursiva que reaccionan en tiempo real a dos tipos de inputs que son la intensidad musical procesada mediante Arduino y la postura corporal del usuario analizada con la librería PoseNet de TensorFlow. La tipografía trasciende el soporte estático para funcionar como una experiencia inmersiva tanto en cartelería física como en entornos web validando la capacidad de la tipografía interactiva para generar nuevos códigos de comunicación visual donde la forma responde al entorno y al gesto del espectador. La arquitectura evidencia la flexibilidad del modelo para gestionar distintas configuraciones de relación. En el caso de #3EFF3F se articula una relación divergente donde convergen valores de entrada sonoros y gestuales para modificar simultáneamente múltiples ejes. El sistema emplea una función normalizadora instrumentada por librerías de procesamiento encargadas de cuantificar magnitudes físicas para adaptarlas al espacio de diseño de la fuente. A partir de estos datos, la función significativa opera bajo lógicas de reactividad, vinculando la intensidad del estímulo con la modificación de diversos ejes de variación. Esta arquitectura confirma el requisito de expandir los procesos de significación, transformando la tipografía estática en una entidad performativa capaz de hacer tangible el ritmo y las dinámicas sonoras de la composición musical.
