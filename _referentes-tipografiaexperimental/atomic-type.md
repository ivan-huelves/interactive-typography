### Descripción
Este proyecto consiste en una herramienta web diseñada para facilitar el aprendizaje de la formulación química en estudiantes de educación secundaria y bachillerato. La autora instrumentaliza la tipografía como herramienta pedagógica para la enseñanza de la ciencia, permitiendo visualizar procesos abstractos de combinación de elementos mediante un sistema gráfico interactivo. La propuesta se basa en una tipografía modificada (basada en la tipografía de código abierto Techna Sans) que utiliza las funciones avanzadas de OpenType como motor lógico. Al introducir el símbolo de un elemento químico mediante el teclado, el sistema activa ligaduras estándar para sustituir los caracteres por representaciones atómicas. Asimismo, mediante el uso de alternativas contextuales, el sistema es capaz de generar moléculas complejas cuando el usuario introduce la fórmula correcta, convirtiendo una cadena de texto lineal en una representación gráfica de los enlaces químicos.

La propuesta se vincula con referentes que investigan la función pedagógica de la tipografía, destacando su conexión con *Reactive Typography*. Ambos proyectos coinciden en entender el diseño como un sistema de aprendizaje multisensorial que emplea la asociación visual directa para facilitar la comprensión de conceptos; sin embargo, mientras que el referente se orienta hacia la enseñanza de idiomas mediante ilustraciones, este proyecto aplica una lógica similar para fomentar la alfabetización científica. Por otro lado, el proyecto conecta con *Hypha* en su intersección entre diseño y ciencia, puesto que ambos proyectos recurren a sistemas biológicos o químicos (como el crecimiento de hongos o la formulación molecular) para generar comportamientos tipográficos que expanden la función comunicativa del texto, convirtiéndolo en una representación visual y pedagógica de procesos científicos complejos. También guarda relación con la mecánica de sustitución de *Seen*, donde el reconocimiento de palabras clave activa un cambio morfológico en la fuente.

La arquitectura de Atomic-Type articula el modelo mediante una relación basada en datos textuales que opera bajo una estricta lógica de sustitución. La cadena de caracteres de la fórmula química actúa como valor de entrada, el cual es procesado por el motor de renderizado de texto que, mediante las alternativas contextuales de OpenType, ejerce de función normalizadora al identificar secuencias de caracteres predefinidas. A partir de estos datos, la función significativa ejecuta una lógica de visualización, reemplazando el texto alfanumérico lineal por la representación gráfica bidimensional de la molécula. Este mecanismo no modula un eje continuo, sino que sustituye el signo tipográfico, validando la capacidad del modelo para expandir los procesos de significación en el ámbito pedagógico, transformando la escritura en una interfaz de verificación de conocimientos donde la aparición de la forma visual recompensa la corrección sintáctica del dato científico.
