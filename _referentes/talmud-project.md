### Relevancia
El Talmud Project es un entorno de estudio digital diseñado para visualizar y navegar la compleja red de textos interconectados del Talmud. El sistema presenta simultáneamente tres capas de texto principales (un ensayo de Levinas, el propio Talmud y la Torá) en un mismo espacio visual. Para gestionar la complejidad, utiliza técnicas de enfoque, desenfoque y transparencia, permitiendo que la capa de interés del lector destaque mientras las otras retroceden visualmente al fondo. 

El desarrollo de una interfaz física permite a uno o dos usuarios manipular de forma tangible y directa la relación entre los textos (cambiando su escala, posición o rotación) para apoyar un argumento o una discusión, imitando el modo de estudio colaborativo tradicional del Talmud. 

La relevancia de este proyecto para la investigación reside en que demuestra cómo una interfaz tangible, combinada con una visualización dinámica, puede facilitar la comprensión y el análisis de un corpus textual multidimensional y estratificado. Establece un precedente clave para la visualización de datos textuales complejos y para la creación de entornos de lectura inmersivos y colaborativos. A pesar de que este proyecto es más próximo al diseño editorial que al tipográfico, se incluye en la investigación por la importancia del autor.
