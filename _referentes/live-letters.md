### Relevancia
Live Letters es una investigación que explora la escritura a través del cuerpo, inspirándose en el funcionamiento de una tabla Ouija. El sistema utiliza un modelo de machine learning, entrenado con un set de datos de posturas corporales, para interpretar y predecir qué letra está realizando el usuario en tiempo real.

Su principal aportación es el enfoque en el proceso de decisión inconsciente y la posibilidad de error en la interpretación. A diferencia de otros sistemas de escritura corporal, el interés del proyecto no reside en la precisión, sino en cómo factores externos (ropa, luz, accesorios) pueden *engañar* al modelo y generar resultados inesperados.

La relevancia de este proyecto es que redefine la interacción tipográfica como un diálogo con un sistema interpretativo, en lugar de un control directo. Demuestra cómo el machine learning puede ser una herramienta para introducir la ambigüedad, el azar y la expresión no deliberada en la comunicación textual, dejando espacio para un input más emocional y menos literal.
