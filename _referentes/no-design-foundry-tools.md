### Relevancia
Este proyecto genera una colección de herramientas web (X-Ray, Pan, Rastr, Rotorizer). La plataforma procesa archivos de fuentes subidos por el usuario aplicando filtros para alterar su morfología original.

Su aportación consiste en proponer un sistema generativo que democratiza el diseño de fuentes experimentales. Al permitir la customización mediante parámetros web, el proyecto habilita a usuarios no especializados para intervenir en la forma de la letra y generar versiones inéditas a partir de archivos existentes.

La relevancia para la investigación radica en que, a diferencia de otros referentes limitados a outputs de imagen o vídeo, este sistema devuelve un archivo de fuente funcional (.otf/.ttf). Esto valida el uso del código no solo para la exploración visual, sino como herramienta capaz de generar software tipográfico disponible en otros entornos de diseño.
